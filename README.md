# J*b Application

## About 

Made for Rageclick YSWS from Hackclub, this is a very minimal J*b Application simulator with impossible ui. For example, the SSN is created from reading the hex code of the color you picked. 

## Features
Impossible UI features such as:
* First & Last name dropdown
* Exponential slider for phone number
* Color picker to Social Security Number
* Sliders for Male and Female for gender
* Weird and restrictive answers to random questions
* Submit button that won't let you submit until everything is filled out
